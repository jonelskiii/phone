Tutorial Windows

Desktop
  1. Tekan ⊞ Win + R → buka Run
  2. Ketik C:\Windows\System32\drivers\etc
  3. Buka file hosts dengan Notepad (Admin)
  4. Tambahkan host → Simpan → Buka Growtopia!
